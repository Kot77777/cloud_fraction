Функция считывает данные об облачности. Данные прдставлены в виде 366 файлов, в которых хранятся результаты дневной облачности. 

Данные в файле записаны следующим образаом\+:
\begin{DoxyItemize}
\item 1 строка -\/ номер дня начиная с нуля. Благодаря этой информации можно получить индекс, начиная с которого будут заполняться данные в вектор за один день в году\+: index = 360 \texorpdfstring{$\ast$}{*} 180 \texorpdfstring{$\ast$}{*} n, где n -\/ номер дня.
\item 2 строка -\/ 360 \texorpdfstring{$\ast$}{*} 180 значений облпчности (в промежутке\+: \mbox{[}0; 1\mbox{]}), вычисленные на сетке с длиной и шириной кадрата в 1 градус. Данные записаны через пробел. Эти значения записываюся в вектор, начиная с индекса\+: index.
\end{DoxyItemize}
